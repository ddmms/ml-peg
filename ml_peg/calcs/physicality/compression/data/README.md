# Compression Benchmark Reference Structures

Place reference crystal structure files here (``.xyz``, ``.extxyz``, or ``.cif`` format).

Each file should contain a single periodic structure with a unit cell defined.
The filename stem is used as the structure label in results and plots.

Example files:
- ``NaCl.xyz``
- ``Si_diamond.cif``
- ``Cu_fcc.extxyz``
